# Landing page O Natural

Site completo em um único arquivo: `index.html` (imagens, fontes e scripts já embutidos).

## Como subir no seu repositório (petterbolt/onattural)

1. Abra https://github.com/petterbolt/onattural
2. Clique em **Add file → Upload files**
3. Arraste o arquivo `index.html` desta pasta
4. Clique em **Commit changes**

## Como publicar (GitHub Pages)

1. No repositório, vá em **Settings → Pages**
2. Em "Branch", selecione **main** e pasta **/ (root)** → **Save**
3. Aguarde 1–2 minutos. O site ficará no ar em:

   **https://petterbolt.github.io/onattural/**

## Atualizações

Quando a página for alterada aqui no projeto, gere um novo `index.html` e repita o upload (o GitHub substitui o arquivo antigo).
